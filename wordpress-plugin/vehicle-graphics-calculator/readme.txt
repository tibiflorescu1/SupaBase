Calculator pentru prețuri grafică vehicule cu integrare WooCommerce.

== Description ==

Vehicle Graphics Calculator este un plugin WordPress care permite clienților să calculeze prețurile pentru servicii de grafică vehicule și să adauge produsele personalizate în coșul WooCommerce.

**Funcționalități principale:**

* Calculator interactiv pentru prețuri grafică vehicule
* Gestionare vehicule, categorii, acoperiri și opțiuni extra
* Materiale de print și laminare cu calcul procentual sau sumă fixă
* Opțiune print cu alb
* Integrare completă cu WooCommerce
* Interfață admin pentru gestionarea datelor
* Shortcode pentru afișare pe orice pagină
* Design responsive și modern

**Utilizare:**

Folosește shortcode-ul `[vehicle_calculator]` pentru a afișa calculatorul pe orice pagină sau articol.

**Opțiuni shortcode:**
* `show_title="false"` - ascunde titlul calculatorului
* `theme="custom"` - aplică o temă personalizată

**Exemplu:**
`[vehicle_calculator show_title="false" theme="custom"]`

== Installation ==

1. Încarcă folderul plugin-ului în directorul `/wp-content/plugins/`
2. Activează plugin-ul din meniul 'Plugins' din WordPress
3. Configurează datele din meniul 'VG Calculator' din admin
4. Adaugă shortcode-ul `[vehicle_calculator]` pe pagina dorită

== Frequently Asked Questions ==

= Este necesar WooCommerce? =

Nu este obligatoriu, dar pentru funcționalitatea de coș de cumpărături, WooCommerce trebuie să fie instalat și activ.

= Cum adaug vehicule noi? =

Mergi la 'VG Calculator' > 'Vehicule' din meniul admin și folosește formularul pentru a adăuga vehicule noi.

= Pot personaliza aspectul calculatorului? =

Da, poți modifica stilurile CSS din fișierul `assets/calculator.css` sau adăuga CSS personalizat în tema ta.
